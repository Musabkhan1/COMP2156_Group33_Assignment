COMP 2156 – DevOps for System Administration is a course that explores the principles and practices of DevOps in modern IT environments. It covers essential topics such as automation, configuration management, continuous integration/continuous deployment (CI/CD), and infrastructure as code. Students gain hands-on experience with tools like Git, Docker, Ansible, and cloud platforms to streamline system administration tasks and improve deployment efficiency. By the end of the course, learners develop the skills to manage and automate IT infrastructure, enhancing collaboration between development and operations teams for faster and more reliable software delivery.
COMP 2156 – DevOps for System Administration is designed to equip students with the skills needed to integrate DevOps methodologies into system administration. The course focuses on automation, version control, containerization, cloud computing, and CI/CD pipelines, ensuring seamless software deployment and infrastructure management.
